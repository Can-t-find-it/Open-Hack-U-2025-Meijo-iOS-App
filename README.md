# 📱 SnAIl

## 🌟 概要
PDFなどのファイルから問題集を自動生成できるアプリ
---

## 🛠 使用技術
- クライアントサイド: **SwiftUI**  
- サーバサイド: **Go, Gin**  
- データベース: **PostgreSQL**  
- インフラ: **AWS, Docker**
